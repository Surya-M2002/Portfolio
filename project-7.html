Frontend Development: Vaanavil Calibration Website

This project involved building a professional, responsive website for Vaanavil Calibration, a technical services company.

The Goal: To build a clean, fast-loading, and professional website to showcase their calibration services, establish credibility, and generate leads.

My Role: I fully redesigned the website’s UI from scratch and updated the entire site using pure HTML, CSS, and JavaScript. I translated the design into a clean, modern interface, ensured full responsiveness across all devices, improved layout structure, and enhanced user experience with subtle animations and better content organization. I also optimized the website for performance and usability.
